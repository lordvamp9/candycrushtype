# Retro Candy Crush - VHS Edition

Welcome to the Retro Low-Poly Candy Crush game! This game combines classic Match-3 gameplay with a nostalgic aesthetic.

## Controls

- **Mouse Left Click**: Select a candy. Click an adjacent candy to swap.
- **Mouse Drag**: Drag a candy to an adjacent slot to swap.
- **ESC**: Pause the game / Open Pause Menu.
- **ENTER**: Resume game from Pause Menu.

## How to Play

1.  **Start Game**: Click "START GAME" on the main menu.
2.  **Objective**: Match 3 or more candies of the same color/shape in a row or column to clear them and earn points.
3.  **Levels**: There are 5 levels. Each level has a higher score requirement and less time to complete.
    - Level 1: 60 Seconds
    - Level 2: 50 Seconds
    - Level 3: 40 Seconds
    - Level 4: 30 Seconds
    - Level 5: 20 Seconds
4.  **Win**: Reach the score threshold before time runs out to advance to the next level.
5.  **Game Over**: If time runs out before reaching the target score, the game ends.

## Features

- **VHS Filter**: Authentic retro visual effects including scanlines, noise, and chromatic aberration.
- **Low Poly Art**: Candies are rendered as geometric shapes using CSS.
- **Audio**: Procedurally generated retro chiptune music and sound effects.
- **Visual Effects**: Particle explosions when matches are made.
- **Goals**: Clear score goals for each level.
- **Victory**: Special victory screen upon completing all 5 levels.
- **Menus**: Fully functional Start, Pause, Game Over, and Level Complete menus.

Enjoy the retro vibes!
